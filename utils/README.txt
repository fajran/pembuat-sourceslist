This is a tool for converting Ubuntu mirrors list page at Launchpad [1] to
a JSON format.

In order to use this, you have to save the page [1] to a file named
"archive.html". After that, open "convert.html" and press "Convert" button.
Save the JSON text into mirrors.json under data sub directory.

[1] https://launchpad.net/ubuntu/+archivemirrors
